实验室三篇论文被国际顶级会议CVPR2018录用

2018-04-07

实验室CVPR论文发表喜获丰收。博士生周彦钊、博士生万方、硕士生邓维建撰写的三篇论文被CVPR2018录用。其中周彦钊的论文被评为“Spotlight”（亮点论文）。祝贺他们！

CVPR全称IEEE Conference on Computer Vision and Pattern Recognition （IEEE国际计算机视觉与模式识别会议）是世界三大顶级计算机视觉会议（ICCV, CVPR和ECCV）之一，每年举行一次。今年会议收到来自上百家世界级名校、研究机构和企业的3300多篇稿件，最终仅接收957篇 (录用率约29%)，而“Spotlight”约为6.8%。今年我实验室共投稿3篇，全部录用，中稿篇数之多和比例之高在国内研究机构尚不多见。

被接收的三篇论文内容如下：

周彦钊、朱艺, Weakly Supervised Instance Segmentation using Class Peak Response (Spotlight)

现有的目标实例分割算法依赖于像素级别的训练数据，标注成本高且训练开销大。本方法利用深度卷积过程中产生的类别峰值响应，从仅采用图片标签训练的分类网络中挖掘出目标级别的精细视觉线索（峰值响应图PRM），在多个评测上取得State-of-the-art性能，并首次在图片级别监督的实例分割任务上报告结果。


万方、魏朋旭, Min-Entropy Latent Model for Weakly Supervised Object Detection, IEEE CVPR, 2018.

MELM针对传统弱监督目标检测框架定位随机性强的问题，结合目标候选区域的分布特征，创新性的使用最小熵隐变量模型，通过分层的方式最小化定位熵，并提出循坏学习的方式求解模型，提高定位稳定性，并能抑制背景噪声，该模型在多个极具竞争性的公开数据集上取得了State-of-The-Art性能。



邓维建, Image-Image Domain Adaptation with Preserved Self-Similarity and Domain-Dissimilarity for Person Re-identification

论文针对行人再识别的跨域场景提出了”Learning via Translation”的算法模型。首先，模型通过无监督且保相似性的方式将有标签的源域数据迁移到目标域上，其次利用迁移后的数据训练专属于目标域的模型。该模型在两个大数据集合（Market1501和DukeMTMC-reid）的跨域场景评估上取得了state-of-the-art的性能。

模式识别与智能系统开发实验室成立于2006年，主要从事机器学习、图像和视频处理、无线网络优化、智能感知系统等领域的研究。实验室研究人员由焦建彬教授、叶齐祥教授、韩振军副教授、秦飞副教授及16名研究生组成，承担了国家“973计划”、中科院“百人计划”、国家自然基金重点及面上项目、中科院电子所、北京市科委等单位课题。
